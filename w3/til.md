## git 복습
git add -A : 현재 폴더 뿐 아니라 모든 것 add
git init 주의사항 : git 로컬 저장소 내에 또다른 git 로컬 저장소를 만들지 말 것

 git add -p :변경사항을 보면서 add 할 지 말 지 정할 수 있음
 git stash : 작업중인 코드를 창고로 보낸다(git stash pop으로 다시 꺼낼 수 있음)
